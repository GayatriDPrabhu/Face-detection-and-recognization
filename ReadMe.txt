README

1. Copy the path for the .xml file required for the CascadeCassifier
2. Copy the path for the .csv file required for the read_csv function
3. Run the program to blur the required face.

Imported files are as follows:
1. opencv2/core/core.hpp
2. opencv2/highgui/highgui.hpp
3. opencv2/imgproc/imgproc.hpp
4. opencv2/objdetect/objdetect.hpp
5. opencv2/face.hpp
